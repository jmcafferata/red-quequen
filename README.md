# Red Quequén Information Bot
## Features
- Get information about the town of Quequén
- Input new information into the system
- Responds in a friendly Argentine dialect

## Installation and Setup
1. Clone the repository:
```bash
git clone https://github.com/yourusername/red-quequen.git
```
2. Change directory to the project folder:
```bash
cd red-quequen
```
3. Install required dependencies:
```
pip install -r requirements.txt
```
4. Set up your OpenAI API key:
```arduino
export OPENAI_API_KEY=your_api_key
```
5. Run the Flask app:
```
python app.py
```
6. Visit http://127.0.0.1:5000/ in your browser to access the application.
